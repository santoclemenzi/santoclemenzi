### :computer: Small web apps I built for fun and learning :

- [Oggi in TV](https://oggiintv.vercel.app/) I like to see TV guide, even if I don't watch TV that much. I really like the TV guide format some smart TV have so I decided to replicate that format with this small Flask PWA.
- [CloudPlayer](https://cloudmusicplayer.vercel.app/) CloudPlayer is the answer to a simple need: playing music from your various cloud accounts with a single player. PWA with just React and Tailwind CSS. No remote database, everything lives in your browser.
- [Digital Business Card](https://farmaciasantacroce.vercel.app/) this started as a PoC and then I think it's actually currently used. It's a small digital business card with a sharing feature through QR code and NFC. React, Tailwind CSS, Supabase.
